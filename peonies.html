PISTIL

R1: 10sc in mr, slst flo
R2: flo: ([7ch, slst], sk, slst)*5, slst blo, ch
R3: blo: 10sc inc, flo slst
R4: flo: ([8ch, slst], sk, slst)*10, ch, coy

PETAL

Inner Petals
R1: blo: 20sc, flo slst
R2: flo: 7ch, from 2nd st, work as: (4dc, sc, flo sc in n st)
R3: 2ch, from 2nd st, slst
R4: 2ch, from 4th st, work as: (3dc, 2sc, flo sc in n st)
R5: 2ch, from 2nd st, slst
R6: 2ch, from 4th st, 3dc, hdc, sc, flo slst in n st
Repeat R2 - R6 4 more times
blo slst in 1st st, ch
Small Petals
R1: blo (3sc, inc) *5, flo slst
R2: flo: 8ch, from 2nd st, work as: (5dc, sc, flo sc in n st)
R3: 2ch, from 2nd st, slst
R4: 2ch, from 4th st, work as: (4dc, 2sc, flo sc in n st)
R5: 2ch, from 2nd st, slst
R6: 2ch, from 4th st, work as: (4dc, 2sc, flo sc in n st)
R7: 2ch, from 2nd st, slst
R8: 2ch, from 4th st, 4dc, hdc, sc, flo slst in n st
flo slst in n st
repeat R2-R8
blo slst in 1st st, ch
Medium Petals
R1: blo (4sc, inc) *5, flo slst, 3 flo slst
R2: flo: 10ch, from 2nd st, work as: (6dc, 2hdc, flo sc in n st)
R3: 2ch, from 2nd st, slst
R4: 2ch, from 4th st, work as: (5dc, 3sc, flo sc in n st)
R5: 2ch, from 2nd st, slst
R6: 2ch, from 4th st, work as: (5dc, 3sc, flo sc in n st)
R7: 2ch, from 2nd st, slst
R8: 2ch, from 4th st, work as: (5dc, 3sc, flo sc in n st)
R9: 2ch, from 2nd st, slst
R10: 2ch, from 4th st, 5dc, hdc, 2sc, flo slst in n st
flo slst in n st
repeat R2-R10
Outer Petals
R1: blo: 30sc, flo slst, 3 flo slst
R2: flo: 11ch, from 2nd st, work as: (7dc, 2sc, flo sc in n st)
R3: 2ch, from 2nd st, slst
R4: 2ch, from 4th st, work as: (6dc, 3sc, flo sc in n st)
R5: 2ch, from 2nd st, slst
R6: 2ch, from 4th st, work as: (6dc, 3sc, flo sc in n st)
R7: 2ch, from 2nd st, slst
R8: 2ch, from 4th st, work as: (6dc, 3sc, flo sc in n st)
R9: 2ch, from 2nd st, slst
R10: 2ch, from 4th st, 6dc, hdc, 2sc, flo slst in n st
flo slst in next st
repeat R2-R8

FLOWER SEPAL

R1: change to green yarn, blo: 30sc, flo slst
R2: 3ch, from 2nd st, work as: sc, flo sc in n st
R3: 3ch, from 2nd st, work as: 3sc, flo sc in n st
R4: 5ch, from 2nd st, 4slst, work as: 3sc, flo sc in n st
R5: ch, from 2nd st, 2slst, work as: sc, flo sc in n st
R6: ch, from 2nd st, slst, flo slst in n st
flo slst in n st
repeat R2-R6
R7: blo: 30sc, slst in 1st st, ch
R8: 15dec, insert wire and stuffing
R9: 7dec, sc
